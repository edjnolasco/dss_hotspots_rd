# DSS_Hotspots_RD v1.0.0

Primera versión estable del sistema DSS para priorización territorial de hotspots viales en República Dominicana.

## Componentes principales
- Visualización geoespacial interactiva
- Ranking Top-K de provincias
- Resumen ejecutivo automático
- Exportación profesional
- Arquitectura modular mantenible
- Modo presentación para demostraciones ejecutivas

## Alcance de esta versión
Esta release establece la base estable del sistema y prepara el proyecto para evolución hacia analítica predictiva, reglas más sofisticadas y fortalecimiento de validación operacional.
